# Library Management System

## Project Overview
This project is a Library Management System implemented in Java.
It demonstrates object-oriented programming concepts such as classes,
objects, inheritance, and encapsulation.

The system manages library items like books and provides basic operations.

## Technologies Used
- Java
- IntelliJ IDEA

## How to Run
1. Open the project in IntelliJ IDEA
2. Run LibraryManagementSystem.java
